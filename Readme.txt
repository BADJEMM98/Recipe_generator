Ce code a pour but de scraper les recettes de marmiton, il génère un fichier csv à la fin de son exécution, plus de 40 milles recettes seront créées.

Etapes à suivres :
1- Création d'un environnement virtuel à l'aide de la commande suivante : 
    python -m venv nlp_env

2 - Activer l'environnement virtuel à l'aide de la commande suivante : 
    .\nlp\Scripts\activate

3- Installer les librairies du requirement à l'aide de la commande suivante :
    pip install -r requirements.txt

4- Lancer le scraping à l'aide de la commande suivante :
    python scraping.py
